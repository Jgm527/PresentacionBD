Automatismos
Trigger post-vuelo

Horas acumuladas
sin intervención

Al finalizar un vuelo, un trigger AFTER UPDATE suma automáticamente las horas de vuelo a cada tripulante.

UPDATE empleados_aire
SET horas_realizadas = horas_realizadas + duracion
FROM vuelos WHERE NEW.estado = 'FINALIZADO'

Trigger cancelación

Reembolso masivo
en cascada

Al cancelar un vuelo, todos sus billetes pasan a REEMBOLSADO sin que la aplicación mueva un dedo.

UPDATE billetes
SET estado = 'REEMBOLSADO'
WHERE idvuelo = NEW.idvuelo
AND estado = 'VIGENTE'
